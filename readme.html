Prerequisites

** work in progress **


1. cvs executable installed and on system path variable.
2. 1.3.1 level jre installed and jre/bin directory added to system path variable.
3. 1.4 level JDK installed (for generating javadoc with working links)
4. (on non Windows systems only) - a 1.3.1 level windows java runtime environment.
5. Info-Zip zip and unzip executables on system path.
6. (for Mac OSX only) tar and gzip
7. org.eclipse.releng.basebuilder and org.eclipse.releng.eclipsebuilder checked out into a directory

cd to org.eclipse.releng.eclipsebuilder directory and execute the following command:

java -cp ../org.eclipse.releng.basebuilder/startup.jar org.eclipse.core.launcher.Main -application org.eclipse.ant.core.antRunner -buildfile build.xml -Dcomponent=sdk | platform | jdt | sdk.examples | sdk.tests [-Dconfigs=<configuration list>] [javacfailonerror=true |false ] [javacVerbose=true | false][zipargs=<arguments to zip executable>] [ -Dbootclasspath=<absolute path to rt.jar to compile against>]

 

examples:

java -cp ../org.eclipse.releng.basebuilder/startup.jar org.eclipse.core.launcher.Main -application org.eclipse.ant.core.antRunner -buildfile build.xml -Dcomponent=sdk -Dconfigs=linux,motif,x86 -Dzipargs=-y -Dbootclasspath=/builds/windows/jre/lib/rt.jar
